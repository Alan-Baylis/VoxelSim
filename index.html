VoxelSim by N3X15

OpenSim. With Voxels.

VoxelSim is OpenSim (an OpenSource BSD-licensed Second Life simulator) heavily modified to use voxels instead of a heightmap terrain.

Dependencies

mono / .NET 3.5 nant

Install

Compile and run from bin.

License
